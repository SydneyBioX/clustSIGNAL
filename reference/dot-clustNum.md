# Neighbour cell initial subcluster label

A function to fetch initial subcluster label of cell.

## Usage

``` r
.clustNum(cell, subClust)
```

## Arguments

- cell:

  a vector of neighbourhood cell indices. The cell indices indicate the
  row number of cells in sample metadata.

- subClust:

  a data frame of initial subcluster labels of each cell in the sample.

## Value

a data frame of initial subcluster labels of neighbourhood cells.
